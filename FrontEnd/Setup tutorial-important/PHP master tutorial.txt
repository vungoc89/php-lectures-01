1. SHIFT + ALT + F => FORMAT CAN CHINH CODE NGAY NGAN
2. CTRL + SHIFT + down-arrow => COPY 1 DONG/1 DOAN CODE
3. CTRL + H => FIND AND REPLACE one/all a word
4. CTRL + SHIFT + click => multiple cursor
5. CTRL + E => delete a line/multiple-line
6: CTRL + J => chọn một loạt từ giống nhau
